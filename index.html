Understanding The Css Box Models

Let's Begin With The Basics

Every Html Element is inside a rectangular box.

Hello World

Which means website is made up of little rectangular boxes.

Understanding The Box Model concept is very helpful when designing a website with CSS.

What Is The Box Model ?

The box model is a term used to describe the multiple boxes that wrap around a single Html element.

The box model consists of:

  • Margin - The content of the box, which include text and images.
  • Border - A border that goes around the padding and content.
  • Padding - Clears an area around the content. The padding is transparent.
  • Content - The content of the box, where text and images appear.
